Air Quality Monitoring and Alert System

The Air Quality Monitoring and Alert System is an IoT-based project designed to detect and alert users about harmful air conditions in real time. Built using Arduino, NodeMCU, and the MQ-135 gas sensor, it measures pollutant levels and displays live AQI data on an LCD screen. When air quality drops below safe levels, a buzzer triggers immediate alerts for safety. The system also uploads data to ThingSpeak Cloud for storage, visualization, and analysis. This project promotes environmental awareness and safety by helping users track and respond to air pollution effectively.
